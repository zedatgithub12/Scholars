Showcase your WordPress posts in beautiful way for elementor page builder.

== Description ==

Blogmentor is an elementor add-on to showcase your WordPress posts in beautiful way. This is an simple and flexible way to decorate your WordPress posts with the help of Elementor page builder.

[View Demo](https://elementor.auburnforest.com/blogmentor/)

### 3 Blog Posts Layouts

We have designed 3 of the most useful blog layouts to climb the top of your design capabilities.

1. [Grid Layout](https://elementor.auburnforest.com/blogmentor/grid-layout/) - 2 different styles available
2. [Masonry Layout](https://elementor.auburnforest.com/blogmentor/masonry-layout/) - 2 different styles available
3. [Metro Layout](https://elementor.auburnforest.com/blogmentor/metro-layout/) - 1 style available
4. Many more coming soon ...

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress


== Frequently Asked Questions ==
= Can I add multiple Blogmentor with single page? =
Yes, you can. Blogmemtor plugin gives you the ability to add **multiple Blog Posts on same page**. You just need to drag and drop that module multiple times and select different post styles whatever you want.

=  Can I change color or customize post style as per my wish? =
Yes, You can customize it. We provided all possible options in backend to update it from default blog posts widget options.
